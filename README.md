# api2json-v2

A Vue 3 + TypeScript + Vite web app for loading and browsing API JSON documents.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Naive UI
- GSAP

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production files are generated in `dist/`.
